### 1. 변수 X가 10보다 크고 20보다 작을 때 변수 X를 출력하는 조건식을 완성하라.

**Question**
```js
var x = 15;

//변수 X가 10보다 크고 20보다 작을 때 변수 X를 출력하는 조건식을 완성하라.
if(...){
  console.log(x);
}
```

**Answer**
```js
var x = 15;

if(x > 10 && x < 20){
  console.log(x);
}
```

### 2. for문을 사용하여 0부터 10 미만의 정수 중에서 짝수만을 작은 수부터 출력하시오.

**Question**
```js
0
2
4
6
8
```

**Answer**
```js
for(var i = 0; i < 10; i++){
  if(i % 2 === 0){
    console.log(i);
  }
}
```

### 11.삼각형 출력하기 - pattern 1

다음을 참고하여 *(별)로 높이가 5인(var line = 5) 삼각형을 문자열로 완성하라. 개행문자(‘\n’)를 사용하여 개행한다. 완성된 문자열의 마지막은 개행문자(‘\n’)로 끝나도 관계없다.

**Question**
```js
// 높이(line)가 5
*
**
***
****
*****
```

**Answer**
```js
// 1. for문 하나로 작성
var line = 5;
var star = '';
var tree = '';
var newLine = '\n';
for(i = 0; i < line; i++){
  star += '*';
  tree += star + newLine;
}
console.log(tree);

// 2. 이중 for문으로 작성
var line = 5;
var star = '*';
var tree = '';
var newLine = '\n';
for(i = 0; i < line; i++){
  for(x = 0; x <= i; x++){
    tree += star;
  }
  tree += newLine;
}
console.log(tree);
```

### 12. 삼각형 출력하기 - pattern 2
다음을 참고하여 *(별)로 트리를 문자열로 완성하라. 개행문자(‘\n’)를 사용하여 개행한다. 완성된 문자열의 마지막은 개행문자(‘\n’)로 끝나도 관계없다.

**Question**
```js
*****
 ****
  ***
   **
    *
```

**Answer**
```js
var line = 5;
var star = '*';
var tree = '';
var newLine = '\n';
var space = ' ';
for(i = 0; i < line; i++){
  for(x = 5; x > i; x--){
    tree += star;
  }
  tree += newLine;
  for(y = 0; y <= i; y++){
    tree += space;
  }
}
console.log(tree);
```
